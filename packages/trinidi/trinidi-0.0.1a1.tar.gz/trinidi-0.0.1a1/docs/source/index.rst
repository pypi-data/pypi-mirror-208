TRINIDI Documentation
=====================

`Time-of-Flight Resonance Imaging with Neutrons for Isotopic Density Inference (TRINIDI) <https://github.com/lanl/trinidi/tree/develop>`__ is a Python package for estimating isotopic densities using neutron transmission resonance images, as described in :cite:`balke2023trinidi`.

This package depends on features provided by SCICO :cite:`balke2022scico`, and provides cross section data that is extracted from the `Lib80x <https://nucleardata.lanl.gov/ace/lib80x/>`__ library :cite:`conlin2018release`.



*TRINIDI is currently under construction.*
